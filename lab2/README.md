# Setup

```bash
pip install numpy matplotlib
```

# Usage

Generate data *(once, ~1–2 min)* — creates `data/`:
```bash
python generate_data.py
```

Run tasks:
```bash
python task1_html_tags.py
python task1_array_stats.py
python task1_matrix_mult.py
python task2_transactions.py
```

Run benchmarks — creates `results/` with 4 PNG files:
```bash
python benchmark.py
```

> All scripts must be run from the directory they are located in.
